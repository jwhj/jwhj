bzoj2957题解

Wed 21 September 2016

如果房子i能被看见，且它前面一座能被看见的房子为j，那么就有hj/j<hi/i。

因此能看到的房子的序列的斜率一定是单调递增的。

每S栋房子分成一块，每块维护一个“可视序列”，即假设其他块的楼房均不存在，这一块中能被看到的楼房的序列。

每次修改时暴力维护可视序列，时间复杂度$O(S)$。

每次查询时，维护目前看到的斜率最大的房屋，从左到右扫每个块，二分找出每块的可视序列中第一个斜率比目前看到的斜率最大的房屋更大的房屋，然后更新答案和目前看到的最大斜率。时间复杂度$O(\frac{n}{S}\log S)$。

易得$S=\sqrt{\frac{n\log n}{2}}$时跑得比较快。

bzoj2463题解

Mon 12 September 2016

当n为偶数时，棋盘可以用多米诺骨牌覆盖，所以后手每走一步，先手总能走到骨牌的另一端，因而先手必胜。

当n为奇数时，除去棋盘的右上角，剩下的区域可以用多米诺骨牌覆盖，所以所以先手每走一步，后手总能走到骨牌的另一端，因而先手必败。

代码：

#include <cstdio>
#define ele int
using namespace std;
int main()
{
    while (1)
    {
        ele n;
        scanf("%d",&n);
        if (!n) break;
        if (n&1) puts("Bob"); else puts("Alice");
    }
    return 0;
}
