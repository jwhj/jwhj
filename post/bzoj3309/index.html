bzoj3309题解

Sep 21, 2016 10:02 · 197 words · 1 minute read 题解 bzoj 数学 mobius

不妨设a<=b。

考虑gcd为k的所有(i,j)的贡献。

$$\begin{align} & \sum{i=1}^{a} \sum{j=1}^{b} f((i, j)) \
= & \sum{d=1}^{a} f(d) \sum{i=1}^{a’} \sum{j=1}^{b’} [(i, j)=1] & \left( a’ = \left \lfloor \frac{a}{d} \right \rfloor, b’ = \left \lfloor \frac{b}{d} \right \rfloor \right) \
= & \sum{d=1}^{a} f(d) \sum{i=1}^{a’} \sum{j=1}^{b’} \sum{k|(i, j)} \mu(k) \
= & \sum{d=1}^{a} f(d) \sum{k=1}^{\left \lfloor \frac{a}{d} \right \rfloor} \mu(k) \left \lfloor \frac{a}{kd} \right \rfloor \left \lfloor \frac{b}{kd} \right \rfloor \
= & \sum{T=1}^{a} \left \lfloor \frac{a}{T} \right \rfloor \left \lfloor \frac{b}{T} \right \rfloor \sum_{d|T} \mu(d) f(\frac{T}{d}) \
\end{align}$$

令$g(T)=\sum_{d|T}\mu(d)f(\frac{T}{d})$，$T={p_1}^{a_1}{p_2}^{a_2}\ldots{p_k}^{a_k}$。

令$S={p_1,p_2,\ldots,p_n},A={p_i|a_i=f(T)},B=S-A$。

那么$\mu(d)$不为0当且仅当d为A、B中选出一些数乘起来的积。

如果存在i,j使$a_i\ne a_j$，那么B不为空集。对于从A中选出数的每种选法，从B中选出奇数个数和偶数个数的选法是相同的。所有这些d的$\mu(d)f(\frac{T}{d})$加起来为0.

如果所有$a_i$都相等，那么只有$d=p_1 p_2\ldots p_k$时,$f(\frac{T}{d})=a_1 -1$,否则$f(\frac{T}{d})=a_1$。易得此时$g(T)=(-1)^{k+1}$。

然后就可以做了。

代码：

#include <cstdio>
#include <cstring>
#include <algorithm>
#define ele long long
using namespace std;
#define maxn 10000010
ele n,pcnt,plist[maxn],mu[maxn],g[maxn];
bool flag[maxn];
int main(){
    pcnt=0;
    memset(flag,0,sizeof(flag));
    mu[1]=1; g[1]=0;
    for (int i=2; i<maxn; ++i){
        if (!flag[i]) plist[pcnt++]=i,mu[i]=-1;
        for (int j=0; j<pcnt && i*plist[j]<maxn; ++j){
            flag[i*plist[j]]=true;
            if (i%plist[j])
                mu[i*plist[j]]=-mu[i];
            else{
                mu[i*plist[j]]=0;
                break;
            }
        }
    }
    memset(g,0,sizeof(g));
    for (int i=2; i<maxn; ++i)
        if (mu[i])
            for (ele j=i; j<maxn && g[j]==0; j*=i)
                g[j]=-mu[i];
    g[0]=0;
    for (int i=1; i<maxn; ++i) g[i]+=g[i-1];
    ele T;
    scanf("%lld",&T);
    while (T--){
        ele a,b;
        scanf("%lld%lld",&a,&b);
        ele ans=0;
        if (a>b) swap(a,b);
        for (int i=a; i>0;){
            int j=a/(a/i+1),j1=b/(b/i+1);
            j=max(j,j1);
            ans+=(a/i)*(b/i)*(g[i]-g[j]);
            i=j;
        }
        printf("%lld\n",ans);
    }
    return 0;
}
