bzoj2729题解

Jan 7, 2017 10:02 · 26 words · 1 minute read 题解 bzoj 数学

先把老师都看成男生随便排，再把女生放进去。

如果排出来两个老师不是相邻的，那么女生可以随便放。

如果排出来两个老师是相邻的，那么他们之间应当恰好放入一个女生，把这三人看成一个整体。

这题需要高精度，我比较懒就直接用了python。

代码：

def P(n,m):
	ans=1
	for i in xrange(m):
		ans*=n-i
	return ans
s=raw_input().split(' ')
n=int(s[0])
m=int(s[1])
a1=P(n+3,m)*(P(n+2,n+2)-P(n+1,n+1)*2)
if (m):
	a2=P(n+2,m-1)*P(n+1,n+1)*2*m
else:
	a2=0
print(a1+a2)
