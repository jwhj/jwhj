bzoj2818题解

May 17, 2016 10:02 · 98 words · 1 minute read 题解 bzoj 数学

因为$(x,y)=p$等价于$(\frac{x}{p},\frac{y}{p})$，所以有： $$ans=\sum{i}\sum{1<=j<=\frac{n}{p_i}}{\phi(j)}$$。

先用线性筛求出所有质数。

先让phi[i]=i，然后对于每个质数p，把phi[p]，phi[2p]，…乘上$\frac{p-1}{p}$。

下面开始玄学的时间复杂度分析：

1~n中，质数有$O(\frac{n}{\log n})$个。

让ave{…}代表那个集合里所有数的平均数。

因为$\frac{n}{1}+\frac{n}{2}+\ldots+\frac{n}{n}=O(n\log n)$，所以$ave{\frac{n}{i}}=O(\log n)$。

质数在1~n中，越到后面越稀疏，所以$ave{\frac{n}{p_i}}=O(\log n)$。

所以求$\phi$函数的时间复杂度为$O(n)$。

代码：

#include <cstdio>
#include <cstring>
#define ele long long
using namespace std;
#define maxn 10000010
ele n,pcnt,plist[maxn],phi[maxn];
bool flag[maxn];
int main(){
    scanf("%lld",&n);
    ele tmp=n+10;
    memset(flag,0,sizeof(flag));
    pcnt=0;
    for (int i=2; i<tmp; ++i){
        if (!flag[i]) plist[pcnt++]=i;
        for (int j=0; j<plist[j] && i*plist[j]<tmp; ++j){
            flag[i*plist[j]]=true;
            if (i%plist[j]==0) break;
        }
    }
    for (int i=1; i<tmp; ++i) phi[i]=i;
    for (int i=0; i<pcnt; ++i)
        for (int j=plist[i]; j<tmp; j+=plist[i])
            phi[j]=phi[j]/plist[i]*(plist[i]-1);
    for (int i=2; i<tmp; ++i) phi[i]+=phi[i-1];
    ele ans=0,num=0;
    for (int i=0; i<pcnt && plist[i]<=n; ++i)
        ans+=phi[n/plist[i]],++num;
    printf("%lld\n",ans*2-num);
    return 0;
}
