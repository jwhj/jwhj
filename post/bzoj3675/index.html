bzoj3675题解

Mar 15, 2017 10:02 · 105 words · 1 minute read 题解 bzoj DP 斜率优化

容易证明顺序是无关紧要的，而且允许出现长度为0的序列的话，答案不受影响。

那么可以写出一个$O(n^2k)$的DP方程：

$$dp[i][p]=\max{dp[j][k-1]+s[j](s[i]-s[j])}$$

然后用斜率优化可以优化到$O(nk)$。

这里有一个小问题，斜率优化的时候，斜率函数$\mathrm{slope}(i,j)=\frac{f[j]-f[k]}{s[k]-s[j]}+s[k]+s[j]$在$s[j]=s[k]$时会出现分母为0的情况。不过我们注意到，序列中的0对答案是没有贡献的，所以我们只需要把序列中的0去掉就可以了。注意此时必须允许出现长度为0的序列。

代码：

#include <cstdio>
#include <cstring>
#include <algorithm>
#define ele long long
using namespace std;
#define maxn 100010
ele n,m,k,a[maxn],b[maxn],s[maxn],Q[maxn],buf1[maxn],buf2[maxn],*f,*g;
inline double slope(ele *f,ele j,ele k){
	return (double)(f[j]-f[k])/(s[k]-s[j])+s[k]+s[j];
}
int main(){
	scanf("%lld%lld",&m,&k);
	for (int i=0; i<m; ++i)
		scanf("%lld",b+i);
	n=0;
	for (int i=0; i<m; ++i)
		if (b[i]) a[n++]=b[i];
	s[0]=a[0];
	for (int i=1; i<n; ++i)
		s[i]=s[i-1]+a[i];
	f=buf1; g=buf2;
	for (int i=0; i<n; ++i)
		f[i]=0;
	for (int p=2; p<=k+1; ++p){
		Q[0]=0;
		ele qh=0,qt=0;
		g[0]=0;
		for (int i=1; i<n; ++i){
			while (qh<qt && slope(f,Q[qh],Q[qh+1])<=s[i])
				++qh;
			ele t=Q[qh];
			g[i]=f[t]+(s[i]-s[t])*s[t];
			while (qh<qt && slope(f,Q[qt-1],Q[qt])>=slope(f,Q[qt],i))
				--qt;
			Q[++qt]=i;
		}
		swap(f,g);
	}
	printf("%lld\n",f[n-1]);
	return 0;
}
