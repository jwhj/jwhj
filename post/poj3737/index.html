poj3737题解

May 11, 2016 10:02 · 88 words · 1 minute read 题解 poj 三分 数学

体积V是关于底面半径r的函数，而且还是上凸的单峰函数。

其实如果把表达式写出来，求个导就可以做了，当然这里也可以用三分法。

下面讲一下三分法，以求上凸的单峰函数的最大值为例:

假设已经确定最大值在[l,r]中，取[l,r]的三等分点m1,m2。

如果f(m1) <= f(m2)，最大值在[l,m2]中。

如果f(m1)>f(m2),最大值在[m1,r]中。

然后就可以做了。

这道题比较奇怪，如果用G++交的话过不了。

代码：

#include <cstdio>
#include <cmath>
#define ele int
using namespace std;
const double eps=1e-6,pi=3.1415926535897;
double s;
inline double h(double r){
	double s1=s-pi*r*r;
    double l=s1/pi/r;
    double h=sqrt(l*l-r*r);
	return h;
}
inline double f(double r){
	return pi*r*r*h(r)/3;
}
inline double func(double x){
	x*=100;
	x=(ele)(x+0.5);
	x/=100;
	return x;
}
int main(){
	while (~scanf("%lf",&s)){
		double l=0,r=sqrt(s/pi);
		while (r-l>eps){
			double m1=l+(r-l)/3;
			double m2=r-(r-l)/3;
			double a1=f(m1);
			double a2=f(m2);
			if (a1<=a2) l=m1;
			else r=m2;
		}
		r=(l+r)/2;
		printf("%.2lf\n%.2lf\n%.2lf\n",func(f(r)),func(h(r)),func(r));
	}
	return 0;
}
