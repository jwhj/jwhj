bzoj3994题解

Jun 6, 2016 10:02 · 134 words · 1 minute read 题解 bzoj 数学 mobius

首先有一条神奇的式子：$$d(nm)=\sum{i|n}\sum{j|m}[gcd(i,j)=1]$$

为什么呢？考虑使用归纳法。如果$gcd(n,m)=1$，那么上面那个式子显然是成立的。否则取一个质数$p$使得$p|n$且$p|m$，让$n=n’\times p^{k1}$，$m=m’\times p^{k2}$，就有$d(nm)=(k1+k2+1)d(n’m’)$。

考虑右式，假设原来有一个互质的数对$(i,j)$，当$n$,$m$分别乘上$p^{k1}$和$p^{k2}$时，就多了数对$(ip,j)(ip^2,j)\ldots(ip^{k1},j)(i,jp)(i,jp^2)\ldots(i,jp^{k2})$，右式的值会乘上$(k1+k2+1)$。

所以就有$d(nm)=\sum{i|n}\sum{j|m}[gcd(i,j)=1]$。

那么就有$$\sum{i=1}^n\sum{j=1}^md(ij)=\sum{i=1}^n\sum{j=1}^m\lfloor\frac n i\rfloor\lfloor\frac m j\rfloor[\gcd(i,j)==1]$$

下面进行一些变换： $$\sum{i=1}^n\sum{j=1}^md(ij)=\sum{i=1}^n\sum{j=1}^m\lfloor\frac n i\rfloor\lfloor\frac m j\rfloor\sum_{d|gcd(i,j)}\mu(d)$$

$$\sum{i=1}^n\sum{j=1}^md(ij)=\sum{i=1}^n\sum{j=1}^m\lfloor\frac n i\rfloor\lfloor\frac m j\rfloor\sum_{d|i,d|j}\mu(d)$$

枚举d： $$\sum{d}^{min(n,m)}\sum{i=1}^{n}\sum_{j=1}^{m}\lfloor\frac{n}{i}\rfloor\lfloor\frac{m}{j}\rfloor[d|i][d|j]\mu(d)$$

显然$i$，$j$得是d的倍数：

$$\sum{d}^{min(n,m)}\sum{p=1}^{\lfloor\frac{n}{d}\rfloor}\sum_{q=1}^{\lfloor\frac{m}{d}\rfloor}\lfloor\frac{n}{pd}\rfloor\lfloor\frac{m}{qd}\rfloor\mu(d)$$

预处理出$f(x)=\sum_{i=1}^{x}\lfloor\frac{x}{i}\rfloor$就可以做了。

代码：

#include <cstdio>
#include <cstring>
#include <algorithm>
#define ele long long
using namespace std;
#define maxn 50010
ele pcnt,plist[maxn],mu[maxn],f[maxn];
bool flag[maxn];
int main()
{
    pcnt=0; mu[0]=0; mu[1]=1;
    memset(flag,0,sizeof(flag));
    for (int i=2; i<maxn; ++i)
    {
        if (!flag[i]) plist[pcnt++]=i,mu[i]=-1;
        for (int j=0; j<pcnt && i*plist[j]<maxn; ++j)
        {
            flag[i*plist[j]]=true;
            if (i%plist[j])
                mu[i*plist[j]]=-mu[i];
            else
            {
                mu[i*plist[j]]=0;
                break;
            }
        }
    }
    for (int i=2; i<maxn; ++i) mu[i]+=mu[i-1];
    memset(f,0,sizeof(f));
    for (int i=1; i<maxn; ++i)
        for (int j=i; j>0;)
        {
            ele p=i/(i/j+1);
            f[i]+=(i/j)*(j-p);
            j=p;
        }
    ele T,n,m;
    scanf("%lld",&T);
    while (T--)
    {
        scanf("%lld%lld",&n,&m);
        ele ans=0;
        for (int i=min(m,n); i>0;)
        {
            ele p=m/(m/i+1),p1=n/(n/i+1);
            p=max(p,p1);
            ans+=f[n/i]*f[m/i]*(mu[i]-mu[p]);
            i=p;
        }
        printf("%lld\n",ans);
    }
    return 0;
}
