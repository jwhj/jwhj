bzoj1011题解

May 2, 2017 10:02 · 74 words · 1 minute read 题解 bzoj 数学

这题真是玄妙无比……

题目描述不是很清楚，应该是要求$f_i=\sum_{j=1}^{A\times i} \frac{m_i m_j}{i-j}$

考虑到A很小，$i$和$i-A\times i$之间的差是很小的，特别是对于比较大的i来说。计算$f_i$的时候，把j分成M段，每一段的i-j当成同一个值来算。这样对于小的i就分得细一些，对于大的i分得粗略一些。M的值看着时限调一调就好了。

要注意计算$A\times i$的时候要加上一个eps以免精度误差。

代码：

#include <cstdio>
#include <cstring>
#include <algorithm>
#define ele int
using namespace std;
#define maxn 100010
#define M 2000
#define eps 1e-8
ele n;
double A,m[maxn],a[maxn];
int main(){
	scanf("%d%lf",&n,&A);
	for (int i=0; i<n; ++i) scanf("%lf",m+i);
	a[0]=0;
	for (int i=1; i<=n; ++i) a[i]=a[i-1]+m[i-1];
	for (int i=0; i<n; ++i){
		ele l=1,r=A*(i+1)+eps,L=(i+1)/M+1;
		double ans=0;
		for (int j=l; j<=r; j+=L){
			ele k=min(j+L,r+1);
			ans+=(a[k-1]-a[j-1])/((double)(i+1)-(double)(j+k-1)/2);
		}
		printf("%lf\n",ans*m[i]);
	}
	return 0;
}
