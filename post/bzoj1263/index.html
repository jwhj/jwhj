bzoj1263题解

Sep 17, 2016 10:02 · 22 words · 1 minute read 题解 bzoj 数学

小学奥数+高精度就可以做了。

我偷懒写了python……

代码：

n=int(raw_input())
tmp=n%3
if tmp==0:
	ans=3**(n/3)
elif tmp==1:
	ans=3**((n-4)/3)
	ans*=4
else:
	ans=3**((n-2)/3)
	ans*=2
ans=str(ans)
L=len(ans)
print(L)
if (L>100): ans=ans[:100]
print(ans)
