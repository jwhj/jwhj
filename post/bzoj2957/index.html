bzoj2957题解

Sep 21, 2016 10:02 · 148 words · 1 minute read 题解 bzoj 分块

如果房子i能被看见，且它前面一座能被看见的房子为j，那么就有hj/j<hi/i。

因此能看到的房子的序列的斜率一定是单调递增的。

每S栋房子分成一块，每块维护一个“可视序列”，即假设其他块的楼房均不存在，这一块中能被看到的楼房的序列。

每次修改时暴力维护可视序列，时间复杂度$O(S)$。

每次查询时，维护目前看到的斜率最大的房屋，从左到右扫每个块，二分找出每块的可视序列中第一个斜率比目前看到的斜率最大的房屋更大的房屋，然后更新答案和目前看到的最大斜率。时间复杂度$O(\frac{n}{S}\log S)$。

易得$S=\sqrt{\frac{n\log n}{2}}$时跑得比较快。

代码：

#include <cstdio>
#include <cstring>
#include <cmath>
#include <algorithm>
#define ele int
using namespace std;
#define maxn 100010
#define maxs 1310
#define eps 1e-10
struct block{
    ele n,m;
    double mx,a[maxs],b[maxs];
};
ele n,m,S;
block a[maxs];
inline void get(ele&a){
    char c;
    while ((c=getchar())<'0' || c>'9');
    a=c-'0';
    while ((c=getchar())>='0' && c<='9') a*=10,a+=c-'0';
}
inline double* upper_bound1(double *a,double *b,double c){
    double *tmp=b;
    --a,--b;
    while (a+1<b){
        double *mid=a+((b-a)>>1);
        if (!(*mid>c)) a=mid; else b=mid;
    }
    if (c<*b) return b; else return tmp;
}
int main(){
    get(n); get(m);
    S=sqrt((double)n*log(n)/log(2)/2);
    for (int i=0,p=0; i<n; i+=S,++p){
        a[p].n=min(S,n-i);
        a[p].m=1;
        for (int j=0; j<a[p].n; ++j) a[p].a[j]=0;
        a[p].mx=a[p].b[0]=a[p].a[0];
    }
    while (m--){
        ele x,y;
        get(x); get(y);
        ele i=(x-1)/S,j=(x-1)%S;
        a[i].a[j]=(double)y/x;
        a[i].m=1;
        a[i].mx=a[i].b[0]=a[i].a[0];
        for (int p=1; p<a[i].n; ++p)
            if (a[i].b[a[i].m-1]<a[i].a[p]) a[i].b[a[i].m++]=a[i].a[p],a[i].mx=max(a[i].mx,a[i].a[p]);;
        double mx=-1;
        ele ans=0;
        for (i=j=0; i<n; i+=S,++j){
            ele p=upper_bound1(a[j].b,a[j].b+a[j].m,mx+eps)-a[j].b;
            ans+=a[j].m-p;
            if (p<a[j].m && fabs(a[j].b[p])<=eps) --ans;
            mx=max(mx,a[j].mx);
        }
        printf("%d\n",ans);
    }
    return 0;
}
