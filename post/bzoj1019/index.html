bzoj1019题解

May 16, 2017 10:02 · 67 words · 1 minute read 题解 bzoj DP

让f[i][j]表示在i号柱上有j个盘子，要多少补步才能把它们移到另一个柱子上，g[i][j]表示移到的柱子的编号。

那么f[i][1]和g[i][1]是可以马上确定的。

对于f[i][j]，不妨设x=g[i][j-1]，y=0+1+2-i-x，那么首先经过f[i][j-1]步，上方的j-1个盘子被移动到x号柱上。因为最小的那个盘子刚刚被移动过，接下来必然是最大的那一个盘子被移到y号柱上。

若g[x][j-1]=y，那么接下来f[x][j-1]步就会把那j-1个盘子移动到y号柱上。

若g[x][j-1]=i，那么接下来f[x][j-1]步把j-1个盘子移动到i号柱上，然后最大的盘子被移动到x号柱上，最后经过f[i][j-1]步，所有盘子移动到x号柱上。

发现自己现在非常智障……求第一个xxx的时候要么从前往后扫找到就break，要么从后往前扫。

代码：

#include <cstdio>
#include <cstring>
#include <algorithm>
#define ele long long
using namespace std;
#define maxn 35
ele n,f[3][maxn],g[3][maxn];
char s[10];
bool vis[3];
int main(){
	scanf("%lld",&n);
	f[0][1]=f[1][1]=f[2][1]=1;
	for (int i=0; i<6; ++i){
		scanf("%s",s);
		if (vis[s[0]-'A']) continue;
		vis[s[0]-'A']=true;
		g[s[0]-'A'][1]=s[1]-'A';
	}
	for (int i=2; i<=n; ++i)
		for (int j=0; j<3; ++j){
			ele x=g[j][i-1],y=3-j-x;
			if (g[x][i-1]==y) f[j][i]=f[j][i-1]+1+f[x][i-1],g[j][i]=y;
			else f[j][i]=f[j][i-1]+1+f[x][i-1]+1+f[j][i-1],g[j][i]=x;
		}
	printf("%lld\n",f[0][n]);
	return 0;
}
