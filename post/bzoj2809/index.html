bzoj2809题解

May 28, 2016 10:02 · 153 words · 1 minute read 题解 bzoj 贪心 优先队列 配对堆

由题，管理者必须是所有被派遣者的祖先。

于是我们就得到了这样一个大暴力：

枚举管理者，在以它为根的子树中从小到大选取被派遣者，直到预算被用完。用这个管理者的领导能力*人数去更新答案。

考虑优化一下这个算法。如何在以它为根的子树中从小到大选取被派遣者，直到预算被用完？可以把以它为根的子树中的所有节点放进一个优先队列里，每次删掉花费最多的，直到花费少于预算。

考虑具体实现。枚举管理者的时候，可以从下往上枚举。对于每个点，保存一个指针，指向一个优先队列。一开始每个点指向的优先队列里面只有那个点本身。每次枚举到一个点，就把它的孩子所指向的优先队列合并起来，再把这个优先队列与自己指向的优先队列合并，就可以得到以它为根的子树中所有节点构成的优先队列。

还有点细节自己想吧。

注意到，这里使用到的优先队列需要支持几个操作：合并、删除最大值。所以可以用左偏树、斜堆、配对堆这样的数据结构维护。我写的是配对堆，据说是个很好用的数据结构？

这里有关于配对堆的介绍。orz yjq

代码：

#include <cstdio>
#include <cstring>
#include <algorithm>
#define ele long long 
using namespace std;
#define maxn 100010
struct edge
{
    ele v;
    edge *nxt;
}ep[maxn];
struct phn
{
    ele x;
    phn *son,*nxt;
}pool[maxn];
inline phn* merge(phn *a,phn*b)
{
    if (!a) return b;
    if (!b) return a;
    if (a->x<b->x) swap(a,b);
    b->nxt=a->son;
    a->son=b;
    return a;
}
phn* mp(phn *x){
    if (!x) return NULL;
    if (!x->nxt) return x;
    phn *p=mp(x->nxt->nxt),*q=x->nxt;
    x->nxt->nxt=NULL; x->nxt=NULL;
    return merge(merge(x,q),p);
}
ele n,m,ecnt,sl[maxn],cnt[maxn],c[maxn],L[maxn];
edge *h[maxn];
phn *lk[maxn];
inline void addedge(ele u,ele v)
{
    edge *p=&ep[ecnt++];
    p->v=v; p->nxt=h[u];
    h[u]=p;
}
int main()
{
    scanf("%lld%lld",&n,&m);
    ecnt=0;
    memset(h,0,sizeof(h));
    for (int i=0; i<n; ++i)
    {
        ele b;
        scanf("%lld%lld%lld",&b,c+i,L+i);
        --b;
        if (b>=0) addedge(b,i);
        sl[i]=pool[i].x=c[i];
        pool[i].son=pool[i].nxt=NULL;
        lk[i]=pool+i;
        cnt[i]=1;
    }
    ele ans=0;
    for (int i=n-1; i>=0; --i)
    {
        phn *p=lk[i];
        for (edge *j=h[i]; j; j=j->nxt)
        {
            sl[i]+=sl[j->v];
            cnt[i]+=cnt[j->v];
            p=merge(p,lk[j->v]);
        }
        while (sl[i]>m)
            sl[i]-=p->x,--cnt[i],p=mp(p->son);
        lk[i]=p;
        ans=max(ans,L[i]*cnt[i]);
    }
    printf("%lld\n",ans);
    return 0;
}
