bzoj4318题解

Dec 9, 2016 10:02 · 40 words · 1 minute read 题解 bzoj 数学

假设已经得到了一个确定的01序列，让$L_i$代表以i结尾的连续1的最长长度，比方说在10111这个串里，$L_0=1,L_1=0,L_2=1,L_3=2,L_4=3$。

记$L_i$的期望为$x_i$，$L_i^2$的期望为$yi$。第i位上如果是1，贡献为$(L{i-1}+1)^3-L{i-1}^3=3L{i-1}^2+3L_{i-1}+1$，否则为0。所以答案为$\sum pi(3x{i-1}+3y_{i-1}+1)$。

注意$y_i\neq x_i^2$。而且x和y不必全部求出来，用两个变量迭代就可以了。

代码：

#include <cstdio>
#include <cstring>
#define ele int
using namespace std;
ele n;
double x,x2,ans;
int main(){
    scanf("%d",&n);
    x=x2=ans=0;
    for (int i=0; i<n; ++i){
        double p;
        scanf("%lf",&p);
        ans+=(3*x2+3*x+1)*p;
        x2=(x2+2*x+1)*p;
        x=(x+1)*p;
    }
    printf("%.1lf\n",ans);
    return 0;
}
