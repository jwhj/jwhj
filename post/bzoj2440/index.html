bzoj2440题解

Jun 7, 2016 10:02 · 106 words · 1 minute read 题解 bzoj 二分 数学 mobius

先吐槽一下题意，所有的整数都是$1^2$的倍数。

然而这题显然是不算$1^2$的。所以一个数不被讨厌当且仅当它没有平方因子。

可以二分答案x，统计1~x中有多少个数无平方因子，然后调整二分的左右端点。

这就留下来两个问题：如何统计、如何确定二分上界。

统计的话可以用容斥，1~x中无平方因子的数的个数为：$$\sum_{i=1}^{\sqrt{n}}\mu(i)\lfloor\frac{x}{i^2}\rfloor$$

至于为什么使劲想一想就明白了吧。

如何确定二分上界呢？先让上界是1，用上面的方法检查一下，如果发现1~1中无平方因子的数不到k，就把上界乘2，反复进行这个操作即可。

代码：

#include <cstdio>
#include <cstring>
#define ele long long
using namespace std;
#define maxn 100010
ele pcnt,plist[maxn],mu[maxn];
bool flag[maxn];
inline ele calc(ele n)
{
	ele ans=0;
	for (int i=1; (long long)i*i<=n; ++i)
		ans+=mu[i]*(n/((long long)i*i));
	return ans;
}
int main()
{
	pcnt=0;
	mu[1]=1;
	memset(flag,0,sizeof(flag));
	for (int i=2; i<maxn; ++i)
	{
		if (!flag[i]) plist[pcnt++]=i,mu[i]=-1;
		for (int j=0; j<pcnt && i*plist[j]<maxn; ++j)
		{
			flag[i*plist[j]]=true;
			if (i%plist[j])
				mu[i*plist[j]]=-mu[i];
			else
			{
				mu[i*plist[j]]=0;
				break;
			}
		}
	}
	ele T,k;
	scanf("%lld",&T);
	while (T--)
	{
		scanf("%lld",&k);
		ele L=0,R=1,mid;
		while (calc(R)<k) R<<=1;
		while (R-L>1)
		{
			ele mid=(L+R)>>1;
			if (calc(mid)>=k) R=mid; else L=mid;
		}
		printf("%lld\n",R);
	}
	return 0;
}
