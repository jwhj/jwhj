bzoj2299题解

Dec 12, 2016 10:02 · 74 words · 1 minute read 题解 bzoj 数学

如果已经有了(a,b)，那么(-a,-b)作用就不大了，可以允许(a,b)的个数为负，然后不管(-a,-b)。

这样有效的向量就剩下四个：(a,b),(a,-b),(b,a),(b,-a)。设他们分别使用了t1,t2,t3,t4次，那么有

$$(t1+t2)a+(t3+t4)b=x$$ $$(t3-t4)a+(t1-t2)b=y$$

令$A_1=t1+t2,B_1=t3+t4,A_2=t3-t4,B_2=t1-t2$，那么$A_1,B_2$同奇偶，$A_2,B_1$同奇偶，且

$$A_1a+B_1b=x$$ $$A_2a+B_2b=y$$

若$A_1,B_1$均为偶数，那么有解当且仅当$2(a,b)|x$且$2(a,b)|y$。

如果$A_1$为偶，$B_1$为奇，变形成

$$A_1a+(B_1-1)b=x-b$$ $$(A_2-1)a+B_2b=y-a$$

这样同理就可以做了。

另外两种情况也是类似的做法。

代码：

#include <cstdio>
#include <cstring>
#define ele long long
using namespace std;
ele t,a,b,x,y,d;
ele gcd(ele a,ele b){
    if (!b) return a;
    return gcd(b,a%b);
}
inline bool test(ele x,ele y){
    return x%d==0 && y%d==0;
}
int main(){
    scanf("%lld",&t);
    while (t--){
        scanf("%lld%lld%lld%lld",&a,&b,&x,&y);
        d=gcd(a,b)<<1;
        if (test(x,y) || test(x-a,y-b) || test(x-b,y-a) || test(x-a-b,y-a-b))
            puts("Y");
        else puts("N");
    }
    return 0;
}
