bzoj4403题解

Nov 4, 2016 10:02 · 101 words · 1 minute read 题解 bzoj 数学

令$m=r-l+1$，长度为i，元素大小在l到r之间的单调不降序列的数量等于长度为i，元素大小在0到m-1之间的单调不降序列的数量。

假设这个序列为$a_1,a_2,\ldots,a_i$，令$b_1=a_1,b_j=aj-a{j-1},b_{i+1}=m-1-a_i(j=2,3,\ldots,i)$，那么就有$b_1+b2+\ldots+b{i+1}=m-1$且所有的b非负。

由插板法可得这样的序列个数为$C{i+m-1}^{i}=C{i+m-1}^{m-1}$。

故答案为

$\sum{i=1}^{n}C{i+m-1}^{m-1}\\=\sum{i=1}^{n}C{i+m-1}^{m-1}+Cm^m-1\\=\sum{i=2}^{n}C{i+m-1}^{m-1}+C{m+1}^m-1\\=\sum{i=3}^{n}C{i+m-1}^{m-1}+C{m+2}^m-1\\\ldots\\=C{m+n}^{m}-1$

用lucas定理来做就可以了。

这里需要求1到$10^6+2$的逆元，如果一个个求会超时。

令$s=P/x,t=P\mod x$，那么有$sx+t\equiv 0(\textrm{mod}.P)$，即$-sx\equiv t(\textrm{mod}.P)$。

于是有$-st^{-1}\equiv x^{-1}(\textrm{mod}.P)$，把式子中的s和t替换掉，得到inv[x]=(P-P/x)*inv[P%x]。

代码：

#include <cstdio>
#include <cstring>
#define ele long long
using namespace std;
#define MOD 1000003
#define maxn (MOD+10)
ele n,l,r,m,inv[maxn],fac[maxn],ifac[maxn];
ele calc_inv(ele x){
	if (inv[x]!=-1) return inv[x];
	return inv[x]=(MOD-MOD/x)*calc_inv(MOD%x)%MOD;
}
inline ele C(ele a,ele b){
	if (a<b) return 0;
	return fac[a]*ifac[b]%MOD*ifac[a-b]%MOD;
}
int main(){
	memset(inv,-1,sizeof(inv));
	inv[1]=1;
	for (int i=2; i<MOD; ++i)
		inv[i]=calc_inv(i);
	fac[0]=1;
	for (int i=1; i<maxn; ++i)
		fac[i]=fac[i-1]*i%MOD;
	for (int i=0; i<maxn; ++i)
		ifac[i]=inv[fac[i]];
	ele T;
	scanf("%lld",&T);
	while (T--){
		scanf("%lld%lld%lld",&n,&l,&r);
		m=r-l+1;
		ele a=n+m,b=n,ans=1;
		while (a || b){
			ans*=C(a%MOD,b%MOD);
			ans%=MOD;
			a/=MOD; b/=MOD;
		}
		printf("%lld\n",(ans-1+MOD)%MOD);
	}
	return 0;
}
