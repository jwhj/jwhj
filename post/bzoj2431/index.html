bzoj2431题解

Jun 20, 2016 10:02 · 71 words · 1 minute read 题解 bzoj DP

中考结束后第一题。话说突然间初中就毕业了，感觉还没反应过来。

让f[i][j]代表1~i的排列中逆序对为j的有多少个。

考虑1~i-1的一个排列P，有j’个逆序对，那么加入一个i，最多可增加i-1个逆序对，最少可增加0个。所以加入i以后的排列的逆序对个数为j’~j’+i-1。

所以$f[i][j]=\sum_{k\le j\le k+i-1} f[i-1][k]$。

直接计算这个dp方程是$O(n^3)$的，不过可以维护每个f[i]的前缀和，把转移的时间优化到$O(1)$，此时算法复杂度为$O(n^2)$。

代码：

#include <cstdio>
#include <cstring>
#define ele int
using namespace std;
#define maxn 1010
#define MOD 10000
ele n,k,f[maxn][maxn],s[maxn][maxn];
int main()
{
    scanf("%d%d",&n,&k);
    memset(f,0,sizeof(f));
    memset(s,0,sizeof(s));
    for (int i=1; i<=n; ++i)
    {
        f[i][0]=1;
        for (int j=1; j<=i*(i-1)/2 && j<=k; ++j)
        {
            ele l=j-i+1,r=j;
            if (l>0) f[i][j]=s[i-1][r]-s[i-1][l-1];
            else f[i][j]=s[i-1][r];
            f[i][j]%=MOD;
            f[i][j]+=MOD;
            f[i][j]%=MOD;
        }
        s[i][0]=1;
        for (int j=1; j<=k; ++j) s[i][j]=s[i][j-1]+f[i][j],s[i][j]%=MOD;
    }
    printf("%d\n",f[n][k]);
    return 0;
}
