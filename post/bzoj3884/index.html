bzoj3884题解

Jun 5, 2016 10:02 · 115 words · 1 minute read 题解 bzoj 数学

神犇们的题解里面都说： >不要被无穷个2吓到了

反正我是被吓到了。

为了叙述方便，我们让$u=2^{2^{2\ldots}}$，那么因为2有无穷个，所以$u=2^u$。

设$p=2^k\times q$，这里$q$为奇数。

那么有$$u\mod p=2^u\mod p=2^k(2^{u-k}\mod q)$$

这里$q$是奇数，与2互质，所以有$$2^{u-k}\mod q=2^{(u-k)\mod\phi(q)}\mod q$$

上面那个指数可以递归地算出来。如果某个时刻$p=1$，那么$\mod p$的结果肯定是0，不需要再递归了。

容易证明递归的层数是$O(\log p)$的。欧拉函数不需要预处理，需要的时候直接$O(\sqrt{p})$算出来就可以了。

整个算法的时间复杂度是$O(T\sqrt{p}\log p)$的。

代码：

#include <cstdio>
#define ele int
using namespace std;
inline ele phi(ele x)
{
	ele tmp=x,ans=x;
	for (int i=2; i*i<=x; ++i)
	{
		if (tmp%i) continue;
		while (tmp%i==0) tmp/=i;
		ans/=i;
		ans*=(i-1);
	}
	if (tmp>1) ans/=tmp,ans*=tmp-1;
	return ans;
}
inline ele pow(ele a,ele x,ele MOD)
{
	ele ans,tmp;
	for (ans=1,tmp=a; x; x>>=1,tmp=(long long)tmp*tmp%MOD)
		if (x&1) ans=(long long)ans*tmp%MOD;
	return ans;
}
ele solve(ele p)
{
	if (p==1) return 0;
	ele q=p,k=0;
	while (~q&1) q>>=1,++k;
	ele q1=phi(q);
	ele tmp=solve(q1);
	tmp=(tmp-k%q1+q1)%q1;
	tmp=pow(2,tmp,q);
	tmp=tmp*pow(2,k,p)%p;
	return tmp;
}
int main()
{
	ele T,p;
	scanf("%d",&T);
	while (T--)
	{
		scanf("%d",&p);
		printf("%d\n",solve(p));
	}
	return 0;
}
