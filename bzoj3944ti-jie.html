bzoj3944题解

Sun 22 May 2016

假设有函数$f(n)$和$g(n)$，满足： $$g(n)=\sum_{d|n}f(d)$$ $$F(n)=\sum_{i=1}^n f(i)$$ $$G(n)=\sum_{i=1}^n g(i)$$ 那么有： $$G(n)=\sum_{i=1}^n\sum_{d|i}f(i)$$ 考虑$f(i)$对$G(n)$的贡献，得到 $$G(n)=\sum_{i=1}^n F(\lfloor\frac{n}{i}\rfloor)$$ 于是有 $$F(n)=G(n)-\sum_{i=2}^n F(\lfloor\frac{n}{i}\rfloor)$$。 假设$G(n)$可以在$O(1)$时间内求出，使用散列表记忆化来求$F(n)$，时间复杂度是多少呢？ 考虑到$\lfloor\frac{n}{i}\rfloor$至多有$O(\sqrt{n})$种取值，所以转移时间是$O(\sqrt{n})$的。

状态总数其实也只有$O(\sqrt{n})$个，因为只有$F(\lfloor\frac{n}{k}\rfloor)$会被递归计算到。

所以总的时间复杂度为 $$\sum_{i=1}^n\sqrt{\frac{n}{i}}=O(n^\frac{3}{4})$$

然而这还是太慢了。

考虑给定一个阀值S，用$O(S)$的时间处理出$i\le S$的$F(i)$，只有当$i\gt S$时才递归计算，使用类似上面的方法分析，可以得到S应取$n^\frac{2}{3}$，此时算法的时间复杂度为$O(n^\frac{2}{3})$。

然而这道题还是很卡常数。

实际上，S稍微开大一点，算法似乎运行得更快。本题有多组数据，每组数据算完之后不清空用来记忆化的散列表，程序运行会快很多。加上这些优化之后我才勉强A掉。

这道题如果用map而不是散列表来记忆化，似乎也不会慢。

代码：

#include <cstdio>
#include <cstring>
#include <cmath>
#include <map>
#define ele long long
using namespace std;
#define maxs 4000010
#define S 4000000
ele n,pcnt,plist[maxs],phi[maxs],mu[maxs];
bool flag[maxs];
map<ele,ele> m1,m2;
ele phisum(ele i){
    if (i<=S) return phi[i];
    if (m1.find(i)!=m1.end()) return m1[i];
    ele ans=i*(i+1)/2;
    for (int j=i; j>1;){
        ele r=i/(i/j+1);
        ele tmp=phisum(i/j)*(j-r);
        ans-=tmp;
        j=r;
    }
    m1[i]=ans;
    return ans;
}
ele musum(ele i){
    if (i<=S) return mu[i];
    if (m2.find(i)!=m2.end()) return m2[i];
    ele ans=1;
    for (int j=i; j>1;){
        ele r=i/(i/j+1);
        ele tmp=musum(i/j)*(j-r);
        ans-=tmp;
        j=r;
    }
    m2[i]=ans;
    return ans;
}
int main(){
    memset(flag,0,sizeof(flag));
    pcnt=0;
    mu[1]=1; phi[1]=1;
    for (int i=2; i<maxs; ++i){
        if (!flag[i]) plist[pcnt++]=i,mu[i]=-1,phi[i]=i-1;
        for (int j=0; j<pcnt && i*plist[j]<maxs; ++j){
            flag[i*plist[j]]=true;
            if (i%plist[j])
                mu[i*plist[j]]=-mu[i],
                phi[i*plist[j]]=phi[i]*(plist[j]-1);
            else{
                mu[i*plist[j]]=0;
                phi[i*plist[j]]=phi[i]*plist[j];
                break;
            }
        }
    }
    for (int i=2; i<maxs; ++i) mu[i]+=mu[i-1];
    for (int i=2; i<maxs; ++i) phi[i]+=phi[i-1];
    ele T;
    scanf("%lld",&T);
    while (T--){
        scanf("%lld",&n);
        printf("%lld %lld\n",phisum(n),musum(n));
    }
    return 0;
}
