git使用总结

Sat 13 August 2016

自从去北欧旅游以后A了题都不写题解了……写点东西拯救一下。

git是什么

自己去看wiki。

需要提一下，git与github是两个不同的东西，你可以认为github是一个支持git的网站，当然这样的网站还有很多，如coding，bitbucket。

安装git

Windows

我早忘了，自己上网查一下吧……很容易搜到的。

Ubuntu

sudo apt-get install git

Arch

sudo pacman -S git

使用git

创建软件仓库

自己在网页端建吧……

把git仓库下载到本地

网页端项目页上应该有一个clone的按钮，可以得到项目的https地址，如https://github.com/jwhj/jwhj.git。执行下面的命令。

git clone https://github.com/jwhj/jwhj.git

一个项目可能有多个分支，按照上面的命令，会下载master分支。如果要指定一个分支，如gh-pages，可以使用-b选项。

git clone -b gh-pages https://github.com/jwhj/jwhj.git

上传本地的项目

如果是第一次使用，需要说清楚你是谁：

git config --global user.email "your email"
git config --global user.name "your name"

首先你的项目得是clone下来的……

假设你没有删除项目中的什么文件。

进入项目的文件夹，执行：

git add ./*
git commit -m '此次提交的备注，不能为空，想不到的话可以随便写点东西'
git push

讲个好玩的东西

刚刚说了有些东西要在网页端操作，可如果我用linux，而且没装图形界面，只用tty的话，该怎么办呢？

sudo pacman -S w3m
w3m github.com

看看效果吧。
