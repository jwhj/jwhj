bzoj2463题解

Mon 12 September 2016

当n为偶数时，棋盘可以用多米诺骨牌覆盖，所以后手每走一步，先手总能走到骨牌的另一端，因而先手必胜。

当n为奇数时，除去棋盘的右上角，剩下的区域可以用多米诺骨牌覆盖，所以所以先手每走一步，后手总能走到骨牌的另一端，因而先手必败。

代码：

#include <cstdio>
#define ele int
using namespace std;
int main()
{
    while (1)
    {
        ele n;
        scanf("%d",&n);
        if (!n) break;
        if (n&1) puts("Bob"); else puts("Alice");
    }
    return 0;
}
